# Workout-Rag-Advisor
RAG-powered Workout Advisor built with LlamaIndex + local Ollama LLM. Pulls from training logs, goals, and a free exercise DB to generate 7-day plans focused on explosive power, with explanations. Tackles RAG challenges like injury-safe retrieval, metadata filtering, and grounded reasoning. Purely offline MVP – Python, Chroma vector DB, CLI-first.
